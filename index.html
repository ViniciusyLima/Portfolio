Sobre mim

Desenvolvedor Full Stack

Graduando em Análise e Desenvolvimento de Sistemas, estou entusiasmado para iniciar minha carreira como Desenvolvedor Java Junior ou Web. Desenvolvi habilidades de lógica de programação, controle de versões, trabalho em equipe. Busco uma oportunidade para aplicar e expandir meus conhecimentos, contribuindo para projetos inovadores.
